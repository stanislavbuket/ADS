# Лабораторна робота 2.3

## Мета
Дослідити основні комбінаторні принципи, типи комбінаторних задач (розміщення, перестановка, комбінація) та використання комбінаторних алгоритмів для їхнього розв’язання.

## Завдання (Варіант 4)
- **Рівень 1**: Розв'язати комбінаторну задачу без повторень: "Скільки варіантів розкладу можна скласти на один день, якщо в день викладають три різні дисципліни з 9?". Визначити тип вибірки.
- **Рівень 2**: Розв'язати комбінаторну задачу з повтореннями: "Скільки дев'ятизначних чисел можна сформувати з набору цифр 1, 2, 3, якщо кількість кожної цифри на одиницю більше за її значення (цифра 1 повторюється 2 рази, 2 — 3 рази, 3 — 4 рази)?". Визначити тип вибірки.
- **Рівень 3**: Записати у файл `combinations.txt` повний перелік розміщень, отриманих під час розв’язання задачі першого рівня (усі 504 варіанти).

## Аналіз задач (Варіант 4)

### Рівень 1: Розклад дисциплін
- **Дано**: Загальна кількість дисциплін $n = 9$, кількість пар на день $k = 3$. Дисципліни мають бути різними.
- **Тип вибірки**: Розміщення без повторень ($A_n^k$), оскільки ми обираємо підмножину елементів, де важливий їх порядок (розклад заліків/пар).
- **Розрахунок**:
  $$A_9^3 = \frac{9!}{(9-3)!} = 9 \times 8 \times 7 = 504$$

### Рівень 2: Формування дев'ятизначних чисел
- **Дано**: Цифри 1, 2, 3. Кількість повторень: для "1" — $1+1=2$ рази, для "2" — $2+1=3$ рази, для "3" — $3+1=4$ рази. Загальна кількість позицій $n = 2 + 3 + 4 = 9$.
- **Тип вибірки**: Перестановки з повтореннями ($P_n(n_1, n_2, \dots, n_k)$), оскільки ми переставляємо фіксований набір елементів, серед яких є однакові.
- **Розрахунок**:
  $$P_9(2, 3, 4) = \frac{9!}{2! \times 3! \times 4!} = \frac{362880}{2 \times 6 \times 24} = 1260$$

## Відповіді на контрольні запитання

**1. Що таке комбінаторика? Які існують загальні правила комбінаторики?**
Комбінаторика — це розділ математики, який вивчає комбінації, розміщення, перестановки елементів скінченних множин та методи підрахунку кількості їх можливих варіантів.
Загальні правила комбінаторики:
- **Правило суми**: Якщо об'єкт $A$ можна вибрати $n$ способами, а об'єкт $B$ — $m$ іншими способами, то вибрати "або $A$, або $B$" можна $n + m$ способами.
- **Правило добутку**: Якщо об'єкт $A$ можна вибрати $n$ способами, і після кожного такого вибору об'єкт $B$ можна вибрати $m$ способами, то пару $(A, B)$ можна вибрати $n \times m$ способами.

**2. Як розв’язуються комбінаторні задачі на розміщення?**
Розміщення — це спосіб вибору $k$ елементів із множини з $n$ елементів, де порядок вибору має значення.
- **Без повторень**: $A_n^k = \frac{n!}{(n-k)!}$
- **З повтореннями**: $\bar{A}_n^k = n^k$

**3. Які розв’язуються комбінаторні задачі на перестановку?**
Перестановка — це впорядкування всіх $n$ елементів деякої множини.
- **Без повторень**: $P_n = n!$
- **З повтореннями**: $P_n(n_1, n_2, \dots, n_m) = \frac{n!}{n_1! n_2! \dots n_m!}$, де $n_1 + n_2 + \dots + n_m = n$.

**4. Які розв’язуються комбінаторні задачі на комбінацію?**
Комбінація — це спосіб вибору $k$ елементів із множини з $n$ елементів, де порядок вибору не має значення (тільки склад).
- **Без повторень**: $C_n^k = \frac{n!}{k!(n-k)!}$
- **З повтореннями**: $\bar{C}_n^k = C_{n+k-1}^k = \frac{(n+k-1)!}{k!(n-1)!}$

**5. Як використовується рекурсія для комбінаторних задач?**
Рекурсія використовується для генерації та перебору комбінаторних об'єктів (перестановок, комбінацій, розміщень). Підхід базується на методі backtracking (пошук з поверненням): рекурсивна функція будує комбінацію крок за кроком (наприклад, обирає перший елемент, запускає себе для решти позицій, а після повернення скасовує вибір та переходить до іншого кандидата). Це дозволяє обійтися без складних багатовимірних циклів.
